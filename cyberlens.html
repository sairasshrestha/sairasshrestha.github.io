Lab Name:

Cyberlens

Goal:

To gain root access

Tools Used:

  • Metasploit
  • ffuf
  • nmap
  • searchsploit

Steps:

1. Initial Exploration:

The first thing I did was use nmap on the target IP to see which ports were open. Then I used ffuf to fuzz the directories. I checked port 80 by simply searching the IP in a web browser, and yes, it was available.

I took a closer look at the website and noticed that it was fetching data from another port. I accessed this port and discovered it was running a deprecated version of Apache Tika.

I searched for this vulnerability on searchsploit and found a corresponding exploit in Metasploit. I set up the exploit with all the necessary options and hit exploit. Success! I got a shell.

I navigated through the system and found the user.txt file at /user/Iforgotthenamebutit'snotadministrator/Desktop/user.txt.

2. Privilege Escalation:

I saved the session by pressing Ctrl+Z, then searched for the post/multi/recon/local_exploit_suggester exploit in Metasploit. I set the session and ran it, which found a working vulnerability.

I used this exploit, set the options, and ran it. Boom! I gained admin privileges and accessed the /Users/Administrator/Desktop/admin.txt file.

Conclusion:

BOOM! Another challenge conquered. Cyberlens proved to be a tough nut to crack, but perseverance and the right tools made it possible. Onward to the next!
