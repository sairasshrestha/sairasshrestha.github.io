Viewing Files in Windows Command Prompt

In Windows Command Prompt (CMD), you can view the contents of files and directories using a few different commands. This guide will show you how to use the dir and type commands to accomplish these tasks.

1. Viewing Files and Directories

To view a list of files and directories in the current directory, use the dir command:

dir

To list files in a specific directory, use the dir command followed by the path to that directory:

dir C:\path\to\directory

2. Viewing Hidden Files

If you need to view hidden files, use the /a option with the dir command:

dir /a

3. Displaying File Contents

To display the contents of a file, you can use the type command:

type filename.txt

Replace filename.txt with the path to the file you want to view.

Example

To display the contents of a file named example.txt in the current directory:

type example.txt

If the file is located in a different directory, specify the full path:

type C:\path\to\file\example.txt

4. Viewing Long Files

If the file is too long to fit on one screen, you can use the type command in combination with the more command to paginate the output:

type filename.txt | more

This will let you scroll through the file one page at a time.

Summary

Using the dir and type commands in Windows Command Prompt allows you to view files, directories, and file contents efficiently. The more command is also useful for paging through long files.
